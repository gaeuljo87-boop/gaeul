# gaeul

데이터 사이언티스트용 간단한 포트폴리오 정적 사이트입니다.

로컬에서 미리보기:

```bash
# 저장소 루트에서 간단한 HTTP 서버로 확인
python3 -m http.server 8000
# 또는
python -m http.server 8000
```

브라우저에서 `http://localhost:8000` 열면 `index.html`을 볼 수 있습니다.

배포 제안:

- GitHub Pages에 푸시하여 배포
- Netlify 또는 Vercel로 정적 사이트 자동 배포

다음 단계:

- `assets/`에 프로필 이미지와 프로젝트 스크린샷 추가
- 프로젝트별 노트북과 링크 추가
- GitHub Pages 또는 Netlify에 배포

문의: `gaeul@example.com`
